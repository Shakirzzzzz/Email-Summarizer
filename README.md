# Email Summarizer (Spring Boot)

An AI-powered backend service that connects to a user's email account, retrieves emails, and generates concise summaries using large language models.
The project demonstrates secure backend design with OAuth2 authentication, JWT authorization, and integration with AI services.

This project focuses purely on backend engineering and API design.
## Demo:
https://youtu.be/xlHLRKAw4Ec?si=o4AzdGjZ-kL2tiMD

## Running the Project
### Start:
```bash
sudo docker-compose up
```
### Stop:
```bash
sudo docker-compose down
```


## Project runs at
http://localhost:8080
